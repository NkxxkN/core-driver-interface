# Core APIs
### `/apiversion` GET
Get a list of compatible CDI versions
```
Request header:
- api-key: string

Response body:
{
  versions: string[]
}
```


### `/config` GET
Get path to the loaded config file
```
Request header:
- api-key: string
- cdi-version: string

Request query params:
- pid: string

Response body:
{
  status: "OK",
  path: string
} | {
  status: "NOT ALLOWED"
}
```


### `/hello` GET, PUT, POST, DELETE
Return a simple hello message
```
Response body:
Hello
```
